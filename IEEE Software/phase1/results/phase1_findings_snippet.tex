\subsection{Baseline Performance}

We evaluated four baseline defenses to establish reference points:
\begin{itemize}
  \item \textbf{Signature-Only}: TPR=79.9%, FPR=0.0%, F1=0.888
  \item \textbf{Rules-Only}: TPR=19.8%, FPR=1.1%, F1=0.328
  \item \textbf{NeMo-Baseline}: TPR=13.0%, FPR=0.5%, F1=0.229
  \item \textbf{OpenAI-Moderation}: TPR=12.0%, FPR=1.5%, F1=0.211
\end{itemize}
